## 1. 项目概况

本项是一个基于 Java 后端驱动的策略类(SLG)游戏原型 。玩家通过养成不同文明的武将，配置兵种与装备，在基于数值与性格影响的异步战斗中取得胜利。

## 2. 核心机制：武将与性格

武将作为战斗核心，其性格将直接决定战斗的走向。
### 2.1 性格系统定义

- **战死沙场**：血量归零前绝不撤退，士气惩罚触发阈值极低。

- **贪生怕死**：血量较低时极高概率触发“认输”，导致士兵战损增加。

- **竭尽全力**：标准性格，按常规概率触发判定。

- **叛逃之心**：极端情况下可能导致战斗直接失败。
### 2.2 核心数值公式

- **武将伤害 (Total Damage)** = (装备攻击力 + 基础攻击力) × 当前等级。

- **战损判定逻辑**：

    - **HP < 20%**：开始判定是否“认输”。若成功，士兵攻击力减少 **20%**。

    - **HP < 10%**：开始判定是否“战死”。若阵亡，士兵攻击力减少 **50%**。
## 3. 文明与特种单位

利用开发者的多语言背景，设计四国文明体系
- **中国** ：特种单位：**解放军**（特性：高纪律性，受武将撤退影响较小）。

- **日本** ：特种单位：**忍者**（特性：高暴击，概率绕过前排直取敌方武将） 。

- **韩国** ：特种单位：**龟甲船**（特性：极高防御，适合持久消耗战） 。

- **英国**：特种单位：**长弓兵**（特性：远程优势，拥有先手攻击权） 。
